**Main class:**

```python
class TranscriptProcessor:
    """Parse and clean transcripts from Panopto (VTT, SRT, TXT formats)"""
    
    def __init__(self):
        self.filler_words = {'um', 'uh', 'like', 'you know', 'basically', 'literally', 'actually', 'so', 'just', 'right'}
    
    def parse_transcript(self, file_path: Path) -> str:
        """
        Parse transcript file (VTT, SRT, or TXT) and extract raw text.
        
        Args:
            file_path: Path to transcript file
        
        Returns:
            Raw transcript text (one string)
        
        Raises:
            TranscriptError: If file missing, unreadable, or unsupported format
        """
    
    def clean_transcript(self, raw_text: str) -> str:
        """
        Clean transcript: remove timestamps, metadata, filler words.
        
        Args:
            raw_text: Raw transcript text from parse_transcript()
        
        Returns:
            Cleaned transcript text
        """
    
    def strip_pii(self, text: str) -> str:
        """
        Remove personally identifying information from transcript.
        
        Args:
            text: Cleaned transcript text
        
        Returns:
            Text with PII stripped (student names, emails, etc.)
        """
    
    def process(self, file_path: Path) -> TranscriptResult:
        """
        Full pipeline: parse → clean → strip PII.
        
        Args:
            file_path: Path to transcript file
        
        Returns:
            TranscriptResult with status, cleaned_text, word_count, error (if any)
        """
```

**parse_transcript() implementation:**

VTT format (WebVTT):
```
WEBVTT

00:00:00.500 --> 00:00:07.000
Caption line 1

00:00:07.000 --> 00:00:10.000
Caption line 2
```

Action:
1. Read file with UTF-8 encoding (handle BOM: UTF-8-sig for VTT files)
2. Detect format:
   - VTT: Contains "WEBVTT" header
   - SRT: Contains numeric line numbers + timestamp lines
   - TXT: Plain text (no special handling needed)
3. For VTT/SRT: Extract only the text lines (skip timestamps, line numbers)
4. Join lines with newlines, preserving paragraph structure
5. Return raw text

Error handling:
- FileNotFoundError: "Transcript file not found: {path}"
- UnicodeDecodeError: "Cannot read transcript file (encoding error). Ensure UTF-8 format."
- Empty file: "Transcript file is empty. Check Panopto download."

**clean_transcript() implementation:**

Using standard library `re` module (no extra dependencies):

1. **Remove timestamps:**
   - VTT format: `00:00:00.500 --> 00:00:07.000` (regex: `\d{2}:\d{2}:\d{2}[.,]\d{3}\s*-->\s*\d{2}:\d{2}:\d{2}[.,]\d{3}`)
   - SRT format: `1\n00:00:00,000 --> 00:00:02,000` (regex: `^\d+\n?\s*`)
   - Panopto metadata: `[Speaker Name]` (regex: `\[\w+\s+\w+\]`)

2. **Remove filler words:**
   - Build list of common filler words: um, uh, like, you know, basically, literally, actually, etc.
   - Regex replace each filler: `\b(um|uh|like)\b` (case-insensitive)
   - Result: "um, that's important" → "that's important"

3. **Normalize whitespace:**
   - Replace multiple spaces/newlines with single space (except paragraph breaks)
   - Preserve sentence structure (one sentence per line)
   - Strip leading/trailing whitespace from each line

4. **Remove URLs and metadata:**
   - Remove URLs: `https?://\S+`
   - Remove email addresses: `\w+@\w+\.\w+`
   - Remove timestamps in brackets: `\[\d{2}:\d{2}:\d{2}\]`

5. **Return cleaned text** (string)

**strip_pii() implementation:**

1. **Detect potential student names/emails:**
   - Email pattern: `\w+@\w+\.\w+` (already removed in clean_transcript, but double-check)
   - Common student name patterns: First + Last separated by space (heuristic detection)
   - WARNING: Over-aggressive removal may break content. Use conservative approach:
     - Only remove if matches known pattern (name@domain.com)
     - Remove obvious identifiers: "student", "john.doe@", "[Student Name]"

2. **Return text with PII removed**

Add error handling for edge cases in TranscriptProcessor:

**Handle missing transcript:**
```python
def process(self, file_path: Path) -> TranscriptResult:
    if not file_path.exists():
        return TranscriptResult(
            status="missing",
            error_message="Transcript not found. Manual upload allowed. Place transcript.txt in output folder and re-run."
        )
```

**Handle malformed transcript:**
- If file contains mixed formats (VTT + SRT + TXT), attempt best-effort parsing
- If parsing fails, return partial result with error message
- Example: "Transcript appears corrupted. Extracted {word_count} words. Review and re-upload if needed."

**Validation checks:**
- Empty file after parsing: `word_count == 0` → error "Transcript is empty"
- Extremely short transcript: `word_count < 100` → warning "Transcript very short (< 100 words). Quality may be poor."
- Cleaning removed too much: `cleaned_length < original_length * 0.2` → warning "Aggressive cleaning removed 80% of text. Review for accuracy."

**Error messages for user:**
- "Transcript file not found: {path}. Place {filename} in {output_dir} and re-run."
- "Cannot read transcript (encoding error). Ensure UTF-8 format. Re-download from Panopto if needed."
- "Transcript is empty. Check Panopto download or re-run with --force-redownload."
- "Transcript quality is poor (< 100 words). Verify Panopto captured full lecture."

**Allow manual override:**
Add method:
```python
def process_manual_transcript(self, text: str) -> TranscriptResult:
    """Process manually-provided transcript text (user pastes content)"""
    return TranscriptResult(status="success", cleaned_text=self.clean_transcript(text))
```

This allows user to manually provide transcript if auto-download failed.
